# Overleaf Source

This is the LaTeX source bundle used for the NeuRoute paper.

Important files:

- `Autohash/Autohash_VLDB2026.tex`: main manuscript source.
- `Autohash/appendix_vldb.tex`: appendix source used for the VLDB appendix.
- `Autohash/Appendix.tex`: extended appendix draft.
- `figures/`: paper figures.
- `Autohash/appendix_figures/`: appendix loss-curve figures.
- `references.bib`: bibliography copy at the bundle root for local compilation.

Historical names such as `Autohash` and `SPHash` are retained to keep the Overleaf paths stable. They refer to the same system released as NeuRoute.
